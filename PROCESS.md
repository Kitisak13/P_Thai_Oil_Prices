# กระบวนการทำงาน (Process Workflow)

โปรเจ็คนี้ประกอบด้วย 2 ขั้นตอนหลักในการทำงาน คือ การดึงข้อมูล (Scraping) และ การรวมข้อมูล (Combining)

## 1. การดาวน์โหลดข้อมูลราคาน้ำมันรายวัน

คำสั่งที่ใช้:
```bash
python main.py
```

**หน้าที่ของสคริปต์:**
- จะทำการเข้าไปที่เว็บไซต์ของกรมพลังงาน (EPPO) เพื่อดึงไฟล์ Excel ราคาน้ำมันแบบรายวัน
- ดาวน์โหลดและบันทึกไฟล์ทั้งหมดไว้ในโฟลเดอร์ `Data/`
- ข้ามไฟล์ที่มีอยู่แล้วในระบบเพื่อประหยัดเวลา
- หน่วงเวลาการดึงข้อมูลเพื่อไม่ให้ส่งผลกระทบต่อเซิร์ฟเวอร์ปลายทาง

*(หมายเหตุ: สามารถแก้ไขตัวแปร `start_date` และ `end_date` ในไฟล์ `main.py` เพื่อระบุช่วงเวลาที่ต้องการดึงข้อมูลได้)*

---

## 2. การรวมไฟล์และจัดรูปแบบข้อมูล

คำสั่งที่ใช้:
```bash
python combine_data.py
```

**หน้าที่ของสคริปต์:**
- อ่านไฟล์ `.xls` ทั้งหมดที่มีอยู่ในโฟลเดอร์ `Data/`
- สกัดชื่อผลิตภัณฑ์และคลีนข้อมูลให้เป็นชื่อภาษาอังกฤษที่เป็นมาตรฐาน (เช่น จาก `เบนซิน ออกเทน 95 (ULG 95 RON)` เป็น `ULG 95 RON`)
- แปลงวันที่จากชื่อไฟล์ให้มาเป็นคอลัมน์ `Date` (รูปแบบ YYYY-MM-DD)
- จัดเรียงข้อมูลให้อยู่ในรูปแบบฐานข้อมูล (Database Long Form) โดยมีชื่อบริษัท (เช่น PTT, BCP, Shell) เป็นหัวคอลัมน์
- นำข้อมูลทั้งหมดมาต่อกัน (Append) และส่งออกเป็นไฟล์ `Combined_Oil_Prices.csv`
